## 通用说明

### 基础 URL

```
http://{host}:{port}
```

### 响应格式

**成功响应：**

```json
{
  "code": 0,
  "msg": "操作成功",
  "data": { ... }
}
```

**分页响应：**

```json
{
  "total": 100,
  "rows": [ ... ],
  "code": 0,
  "msg": "查询成功"
}
```

**失败响应：**

```json
{
  "code": 500,
  "msg": "错误信息"
}
```

### 通用请求头

| 参数名 | 类型   | 必填 | 说明         |
| ------ | ------ | ---- | ------------ |
| Cookie | String | 是   | Session 凭证 |
